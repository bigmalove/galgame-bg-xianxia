# 内置图包·仙侠

galgame 界面插件内置背景图包（30 个场景，2K webp）。

## 场景清单

竹林、山巅、仙宫、剑冢、云海、瀑布、洞府、丹房、灵田、古战场、幽谷、天池、藏经阁、演武场、坊市、雪山、桃林、灵泉、祭坛、仙岛、秘境、传送阵、山门、崖底、冰窟、魔渊、锁妖塔、观星台、后山、静室

## 在插件中使用

资源管理器 → 导入 → 内置背景图包，或手动「导入远程链接JSON」填入：

```
https://cdn.jsdelivr.net/gh/bigmalove/galgame-bg-xianxia@v1.1.0/remote_assets.json
```
